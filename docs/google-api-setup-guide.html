1

Google Cloud 프로젝트 생성

  1. 1.
    Google Cloud Console 접속
  2. 2.
    상단 프로젝트 선택 → 새 프로젝트 클릭
  3. 3.
    프로젝트 이름 입력: keai-analytics
  4. 4.
    만들기 클릭 후 프로젝트 생성 완료 대기 (약 30초)
2

API 활성화

① Google Analytics Data API

  1. • Analytics Data API 페이지 접속
  2. • 프로젝트가 keai-analytics인지 확인
  3. • 사용 버튼 클릭

② Search Console API

  1. • Search Console API 페이지 접속
  2. • 사용 버튼 클릭
3

서비스 계정 생성

  1. 1.
    서비스 계정 페이지 접속
  2. 2.
    + 서비스 계정 만들기 클릭
  3. 3.
    서비스 계정 이름: keai-analytics-reader
  4. 4.
    만들고 계속하기 → 역할 선택 생략 → 완료
  5. 5.
    생성된 서비스 계정 클릭 → 키 탭 → 키 추가 → 새 키 만들기
  6. 6.
    키 유형: JSON 선택 → 만들기 → JSON 파일 다운로드됨
4

GA4 속성에 권한 부여

💡 GA4가 없다면?
Google Analytics에서 먼저 GA4 속성을 생성하세요.

  1. 1.
    Google Analytics 접속
  2. 2.
    좌측 하단 ⚙️ 관리 클릭
  3. 3.
    속성 열에서 속성 액세스 관리 클릭
  4. 4.
    우측 상단 + → 사용자 추가
  5. 5.
    이메일 주소에 서비스 계정 이메일 입력 (위에서 기록한 것)
  6. 6.
    역할: 뷰어 선택 → 추가

관리 → 속성 설정 → 속성 ID (숫자만)

5

Search Console에 권한 부여

  1. 1.
    Google Search Console 접속
  2. 2.
    k-eai.kr 속성 선택
  3. 3.
    좌측 메뉴 설정 → 사용자 및 권한
  4. 4.
    사용자 추가 클릭
  5. 5.
    이메일에 서비스 계정 이메일 입력
  6. 6.
    권한: 전체 또는 제한됨 선택 → 추가
